# ChatterBox — Discord Clone

## How to Run

### Requirements
- Node.js 18+
- MongoDB running locally on port 27017

### 1. Start MongoDB
Make sure MongoDB is running on your machine.

### 2. Start the Backend
```
cd server
npm run dev
```
Server runs on: http://localhost:5000

### 3. Start the Frontend
```
cd client
npm run dev
```
App runs on: http://localhost:5173

---

## Tech Stack Used
- ReactJS
- ExpressJS
- Axios
- Socket.io
- MongoDB (Mongoose)
- Pure CSS
